Title:

$pages->pluck($field, $split = null, $unique = false)

----

Excerpt:

Extracts an array of values from each item by field.

----

Return:

array

----

Text:

## Example

```php
<ul>
  <?php foreach($page->children()->pluck('title') as $title): ?>
  <li><?php echo html($title) ?></li>
  <?php endforeach ?>
</ul>
```
